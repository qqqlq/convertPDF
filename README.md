# convertPDF

SharePoint / Microsoft Stream などダウンロード不可の講義動画を、自動キャプチャして PDF に変換する Chrome 拡張機能。

## インストール方法

1. リポジトリをクローン or `git pull`
2. `npm install && npm run build`（Node.js が必要）  
   　または `dist/` フォルダをそのまま使用（ビルド済みを git push してある）
3. Chrome の `chrome://extensions` を開く
4. 右上「デベロッパーモード」をオン
5. 「パッケージ化されていない拡張機能を読み込む」→ `dist/` フォルダを選択

## 使い方

1. SharePoint / Stream の動画ページを開く
2. ツールバーのアイコンをクリックしてポップアップを開く
3. キャプチャ間隔を設定（既定 10 秒）
4. **「録画開始」** を押して動画を再生
5. アニメーション等で取りこぼしが心配な場合は **「今すぐキャプチャ」** を手動で押す
6. 動画終了後 **「録画停止」→「PDF生成」**
7. `slides.pdf` がダウンロードされる

## キャプチャの仕組み

**2 系統のトリガーが同時動作する：**

| トリガー | 間隔 | 用途 |
|---|---|---|
| バイナリカウント差分検知 | 250ms ごとにチェック | アニメーションで数式・箇条書きが追加された瞬間を捉える |
| 定期キャプチャ | N 秒ごと（既定 10 秒） | 板書など緩やかな変化の保険 |
| 手動キャプチャ | ボタン押下時 | 確実に残したい瞬間 |

**PDF 生成時の重複削除（2 段階）：**

1. **dHash**（差分ハッシュ）: 視覚的にほぼ同じフレームを除去
2. **スーパーセット判定**: 「同じスライドにコンテンツが追加されただけ」のフレームを折りたたみ、最終状態だけ残す ※現在チューニング中

## 対応プラットフォーム

- Microsoft SharePoint
- Microsoft Stream
- `*.svc.ms` 配信コンテンツ

## ポップアップの設定項目

| 設定 | 説明 |
|---|---|
| キャプチャ間隔 | 5〜120 秒（板書: 30〜60 秒、アニメーションスライド: 5〜10 秒推奨） |
| 重複判定（ハミング距離）| 詳細設定。0〜64。小さいほど厳しく重複を残す（既定 8） |

## 開発

```bash
npm install
npm run dev    # watch モードでビルド（Chrome に読み込んで開発）
npm run build  # 本番ビルド
```

ビルドツール: Vite + @crxjs/vite-plugin + TypeScript

## 既知の問題・TODO

- **スーパーセット判定の精度**: 段階的に内容が表示されるスライドの折りたたみが不十分な場合がある。閾値パラメータのチューニングが必要。
- **PDF サイズ**: フル解像度 PNG のため枚数が多いと大きくなる（将来的に JPEG オプションを検討）。
- **DRM 動画**: tabCapture でもブラックアウトするため対応不可。
